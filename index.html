⚙ GOOGLE SHEETS SETUP

Connect via Google Apps Script — a free tool built into Google Sheets that bypasses all browser restrictions. One-time setup, 3 minutes.

1
Open Apps Script from your Google Sheet
Open your Google Sheet on a desktop/laptop browser.
Click Extensions in the top menu → click Apps Script.
A new tab opens with a code editor.
2
Paste this code — replace everything in the editor
Select all existing code (Ctrl+A) → delete → paste this:
function doGet(e) { var tab = e.parameter.tab || 'wins'; var ss = SpreadsheetApp.getActiveSpreadsheet(); var sheet = ss.getSheetByName(tab); var data = sheet.getDataRange().getValues(); var csv = data.map(function(r){return r.join(',');}).join('\n'); return ContentService.createTextOutput(csv) .setMimeType(ContentService.MimeType.TEXT); }
3
Deploy as Web App
Click Deploy (top right) → New deployment
→ Click the ⚙ gear icon next to "Select type" → choose Web app
→ Set "Execute as" to Me
→ Set "Who has access" to Anyone
→ Click Deploy → click Authorize access
→ Choose your Google account → click Allow
→ Copy the Web app URL shown (starts with script.google.com)
4
Paste your Web App URL below
Looks like: https://script.google.com/macros/s/ABC123.../exec

💡 Why this works: Apps Script runs on Google's servers and delivers your sheet data directly to the app. It's free, instant, and permanent.
